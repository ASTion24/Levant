# LEVANT 自动化推演引擎 (Levant Engine) v1.0

![Levant Logo](./logo.png)

> **Powered by 絜矩 | AI-Driven Strategy Deduction System**

## 📖 简介 | Introduction

Levant 是一个专为文字游戏、跑团（TRPG）、安科（Anco）及战略游戏推演设计的自动化辅助工具。它结合了现代 LLM（大语言模型）的创造力与严谨的状态机逻辑，帮助推演（GM）高效地管理世界观、实体状态和时间线。

主要功能：
*   **AI 辅助推演**：基于当前世界状态，自动生成下一回合的剧情与数据变更。
*   **可视化战报**：一键生成“终端风格”或“报纸风格”的精美战报海报。
*   **状态管理**：可视化的实体属性、资料库（Lore）及全局变量管理。
*   **本地化部署**：数据完全存储于本地 JSON，安全可控。

## 🚀 快速开始 | Quick Start

本程序已打包，无需安装 Python 环境。

1.  解压压缩包到任意文件夹（请确保路径中不包含中文，以防兼容性问题）。
2.  双击运行文件夹内的 **`Levant自动化推演.exe`**。
3.  程序会自动启动后台服务，并自动打开默认浏览器加载操作界面。
4.  点击右上角的设置（齿轮图标），配置您的 AI API Key（支持 Gemini / OpenAI）即可开始使用。

> **注意**：如果遇到“网络错误”或 AI 无响应，请在设置中开启“本地代理”并检查端口（默认为 7890）。
